### 1.node + npm
#### 1.1 npm 常用命令
    npm init  在项目中引导创建一个package.json文件，包括名称、版本、作者这些信息等
    npm start 启动模块
    npm stop  停止模块
    npm publish 发布模块
    npm ls      查看安装的模块
    npm root    查看包的安装路径
    npm卸载插件：npm uninstall <name> [-g] [--save-dev] 
    npm更新插件：npm update <name> [-g] [--save-dev] 
    npm更新全部插件：npm update [--save-dev] 
    npm帮助：npm help 
    npm查看当前目录已安装插件：npm list
